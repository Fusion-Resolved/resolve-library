Optical Flow Warp

Fusion Effect
● Intermediate 12 Mar 2025

Uses optical flow vectors to warp a source clip, creating a liquid smear without third-party plugins. Works best on clips with strong directional motion. Chain two passes for a more pronounced effect.

Step-by-step
  1. 01Add an OpticalFlow node after your source clip and set Quality to High.
  2. 02Pipe the flow output into DisplaceDistort. Set Channel to Luminance.
  3. 03Adjust X/Y Scale parameters — 0.3 to 0.8 is a good starting range.
  4. 04Add a TimeSpeed node upstream, offset the flow by 1-2 frames for a ghosting look.
Node tree code
OpticalFlow → DisplaceDistort → TimeSpeed → Merge
