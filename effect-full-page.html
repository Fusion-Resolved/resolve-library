About this effect

Optical Flow Warp uses DaVinci Fusion's native OpticalFlow node to generate a per-pixel velocity map from frame-to-frame motion, then feeds that data into a DisplaceDistort node to warp the source clip. The result is a liquid smear that follows actual movement in the footage — no third-party plugins required.

Works best on clips with strong, clean directional motion. A TimeSpeed node upstream offsets the flow source by one to two frames, which adds a ghosting quality to the warp. Chain two full passes for a heavier effect, or blend back to the original at lower opacity for a more subtle look.

Optical FlowWarpFusion DistortNo Plugins
Step-by-step
01Add an OpticalFlow node after your MediaIn and set Quality to High for the cleanest vector map.
02Connect a TimeSpeed node in parallel off the same MediaIn. Set Speed to 0.97 to offset the source by roughly one frame.
03Route both outputs into DisplaceDistort — OpticalFlow into the Disp Map input, TimeSpeed into the Image input. Set Channel to Luminance.
04Adjust X/Y Scale to between 0.3 and 0.8. Values above 1.5 create heavy tearing — use Wrap Mode Smear to handle edge pixels cleanly.
05Feed the result through a ColorCorrector to recover saturation lost in the warp, then merge back over the original plate via a Merge node at your desired opacity.
